# Simple Interest Calculator

This project explains a simple interest calculator.

## Formula

Simple Interest (SI) is calculated using the formula:

\[
SI = \frac{P \times R \times T}{100}
\]

Where:
- **P** = Principal amount (initial amount of money)
- **R** = Rate of interest per year (in %)
- **T** = Time period (in years)

## Example

If:
- P = 10,000  
- R = 5% per year  
- T = 2 years  

Then:

\[
SI = \frac{10{,}000 \times 5 \times 2}{100} = 1{,}000
\]

Total Amount = Principal + Interest = 10,000 + 1,000 = 11,000

## Pseudocode

```text
input P, R, T
SI = (P * R * T) / 100
Amount = P + SI
print SI, Amount
